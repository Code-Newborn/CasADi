## CasADi 是什么？



**[CasADi](https://web.casadi.org/)** 是一个面向 **数值优化与最优控制** 的开源软件库，核心能力是把**数学表达式以符号形式建模**，并**自动生成高效、可微的数值代码**，用于求解各类优化问题（尤其是带约束的非线性优化）。

它本身**不是求解器**，而是一个**建模 + 自动求导 + 代码生成平台**，通常与成熟求解器（如 IPOPT、OSQP 等）配合使用。

## 它解决什么问题（核心价值）?

- **自动求导（AD）**：精确、快速地计算梯度、雅可比、海森矩阵（比数值差分稳定、比手推省力）。
- **符号建模 → 数值执行**：先写“数学式子”，再高效执行数值计算。
- **复杂约束建模**：自然表达非线性、等式/不等式、稀疏结构。
- **代码生成**：可生成 C/C++ 代码，用于实时系统、嵌入式部署。
- **与工业级求解器无缝对接**：把模型“喂给”高性能优化器。

## 能干什么（典型应用场景）?

#### 1) 最优控制 / MPC（模型预测控制）

- 无人机、机器人、车辆轨迹规划
- 工业过程控制（化工、能源）
- 实时控制（配合代码生成）

#### 2) 非线性规划（NLP）

- 工程设计优化
- 参数辨识 / 系统辨识
- 约束最小二乘

#### 3) 动态系统建模

- 常微分方程（ODE）
- 微分代数方程（DAE）
- 灵敏度分析

#### 4) 科研与工程中的“优化内核”

- 不直接写求解器
- 把数学模型稳定、可维护地交给 CasADi

## CasADi vs 其他工具（简要对比）

| 工具            | 特点                          | 适合人群        |
| --------------- | ----------------------------- | --------------- |
| CasADi          | 自动求导 + 代码生成 + NLP/MPC | 控制 / 优化工程 |
| CVX / CVXPY     | 凸优化建模                    | 凸优化为主      |
| SciPy optimize  | 数值优化工具集                | 轻量级          |
| 手写 IPOPT 接口 | 性能高但复杂                  | 底层专家        |
